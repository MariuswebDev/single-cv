# Single-Page CV

Solution for the roadmap.sh Single Page CV project.

Project URL: https://roadmap.sh/projects/single-page-cv

A clean and professional single-page CV/resume template built with HTML and CSS.

## Overview

This project is a minimalist curriculum vitae designed to showcase your professional information in a single, printable page format. It includes sections for contact information, skills, education, and work experience.

## Files

- **cv.html** - The main CV document with semantic HTML structure
- **cv.css** - Styling and layout for the CV
- **README.md** - This file

## Features

- Clean, professional design
- Single-page layout (optimized for print)
- Responsive styling with CSS flexbox
- Organized sections:
  - Personal introduction with contact info
  - Skills
  - Education
  - Work experience

## How to Use

1. Open `cv.html` in your web browser
2. Edit the HTML content to add your personal information:
   - Name and job title
   - Contact details (address, phone, email)
   - Skills
   - Education history
   - Work experience and achievements

3. Customize the CSS in `cv.css` to match your preferred styling

## Printing

To save as PDF:

1. Open `cv.html` in your browser
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Select "Save as PDF" and customize print settings
4. Print to file
